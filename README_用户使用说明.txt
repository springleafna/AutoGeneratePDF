🌟 欢迎使用【AutoGeneratePDF】图形界面版（桌面自动保存）！

📌 使用方法：

1. 双击打开 “AutoGeneratePDF.exe”
   → 弹出窗口，输入你的打印页面网址（如：https://admin.lxl.com/...）

2. 点击 “✅ 开始打印”

3. 程序会自动：
   - 打开浏览器
   - 点击“打印中英文”、“打印英文”、“打印中文”
   - 自动保存三个 PDF 文件

4. 查看结果：
   → 打开你的 **桌面**
   → 找到名为 **AutoGeneratePDF** 的文件夹
   → 进入里面以今天日期命名的子文件夹（如：250914）
   → 你会看到三个文件：
        学习资料_中英文_陈唤唤_250914.pdf
        学习资料_英文_陈唤唤_250914.pdf
        学习资料_中文_陈唤唤_250914.pdf

💡 每天都会自动创建新文件夹（如明天是 250915），不会覆盖旧文件！

❗ 注意事项：
- 请确保电脑已联网
- 不要关闭窗口，直到出现“全部完成”提示
- 如遇问题，请截图发给技术支持
